# **Исследование внутриигровой экономики**

### **Задачи проекта**

- анализ влияния характеристик игроков и персонажей на покупку внутриигровой валюты
- изучение активности игроков при совершении внутриигровых покупок
- выявление ключевых закономерностей в поведении платящих пользователей

### **Описание проекта**

Проведён исследовательский анализ данных онлайн-игры, охватывающий различные аспекты внутриигровой экономики и поведения игроков.

### **Ключевые направления анализа**

- **распределение платящих игроков** по расам персонажей
- **статистика покупок:** стоимость, количество, частота транзакций
- **выявление аномалий** в паттернах покупок
- **анализ популярности** внутриигровых предметов и предложений
- **сегментация игроков** по модели монетизации (частота и размер покупок)

### **Основные выводы**

- определена зависимость между характеристиками персонажа (раса) и покупательской активностью
- выявлены паттерны поведения платящих игроков
- обнаружены аномалии в структуре покупок, требующие дополнительного изучения
- определены наиболее популярные категории внутриигровых товаров

### **Рекомендации**

1. оптимизировать внутриигровую экономику на основе выявленных закономерностей
2. персонализировать предложения для разных сегментов игроков
3. повысить вовлечённость через целевые механики монетизации
4. провести дополнительный анализ выявленных аномалий для понимания их причин
