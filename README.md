# Анализ онлайн-игры

**Задачи проекта:** Изучить влияние характеристик игроков и игровых персонажей на покупку внутриигровой валюты, проанализировать активность игроков при совершении внутриигровых покупок и выявить ключевые закономерности.

**Описание проекта:** Проведён исследовательский анализ данных онлайн-игры, включая распределение платящих игроков по расам персонажей, статистику стоимости и количества покупок, выявление аномалий и популярности предметов. Решены ad hoc задачи по оценке зависимости активности от расы персонажа и сегментации игроков по частоте покупок. Сформулированы выводы и рекомендации для оптимизации внутриигровой экономики и повышения вовлечённости игроков.
